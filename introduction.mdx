# Welcome to The Movie Database API (TMDB)

The TMDB API offers an extensive collection of resources for developers to access movie, TV, and actor-related data. With this API, you can build applications that retrieve and manipulate content from one of the most popular movie and TV databases worldwide.

## Base URL

All requests to the TMDB API should be made to:
```
https://api.themoviedb.org/3
```

## Authentication

### API Key
To interact with the TMDB API, you must first generate an API key. This key will be used to authenticate your requests.

- [Learn more about generating an API key](https://developer.themoviedb.org/reference/authentication-how-do-i-generate-a-session-id)

### Session Management
TMDB provides multiple ways to manage sessions, including guest sessions and fully authenticated user sessions.

- [Creating Guest Sessions](https://developer.themoviedb.org/reference/authentication-create-guest-session)
- [Creating User Sessions](https://developer.themoviedb.org/reference/authentication-create-session)

## API Resources

The TMDB API offers access to the following resources:

### Accounts
Manage and retrieve user account details and preferences.

- [Get Account Details](https://developer.themoviedb.org/reference/account-get-details)
- [Manage Favorite Movies and TV Shows](https://developer.themoviedb.org/reference/account-mark-as-favorite)
- [Add Movies or TV Shows to Watchlist](https://developer.themoviedb.org/reference/account-add-to-watchlist)

### Authentication
Handle API key validation and session management.

- [Validate API Key](https://developer.themoviedb.org/reference/authentication-validate-key)
- [Create Request Token](https://developer.themoviedb.org/reference/authentication-create-request-token)

### Certifications
Retrieve certification ratings for movies and TV shows by region.

- [Get Movie Certifications](https://developer.themoviedb.org/reference/certifications-get-movie-list)
- [Get TV Certifications](https://developer.themoviedb.org/reference/certifications-get-tv-list)

## Getting Started

1. **Sign Up**: Create an account at [TMDB](https://www.themoviedb.org/).
2. **Get an API Key**: Obtain your unique API key from the [API Dashboard](https://www.themoviedb.org/settings/api).
3. **Explore the API**: Review the [TMDB API Documentation](https://developer.themoviedb.org/).

## Rate Limiting
TMDB enforces rate limits to ensure fair usage. For detailed information about rate limits, consult the [Rate Limiting Guide](https://developer.themoviedb.org/docs/rate-limiting).
